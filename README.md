# 🚀 FlowAI – AI Workflow Automation Platform

> **Automate Every Workflow. Powered by AI.**

FlowAI is a premium, modern SaaS landing page for an AI-powered workflow automation platform. Built with **React**, **Vite**, and **Tailwind CSS**, it showcases a sleek, responsive UI inspired by industry-leading products like **Stripe, Linear, Vercel, Notion AI, and OpenAI**.

🌐 **Live Demo:** https://asset-manager--shraddhakps321.replit.app/

---

## ✨ Overview

FlowAI helps businesses automate repetitive tasks, streamline workflows, process documents intelligently, integrate with multiple platforms, and gain real-time AI-powered insights—all through a beautiful and intuitive interface.

This project was built with a strong focus on **modern UI/UX, responsiveness, accessibility, and performance**, making it suitable for frontend competitions and production-ready deployments.

---

## 🌟 Features

- 🎨 Premium dark-themed SaaS UI
- ✨ Glassmorphism & animated gradient effects
- 📱 Fully responsive (Mobile, Tablet & Desktop)
- 🧩 Responsive Bento Grid that transforms into an Accordion on mobile
- 💰 Dynamic pricing section
  - Monthly / Annual billing toggle
  - Multi-currency support (USD, INR, EUR)
  - Annual pricing calculated dynamically
- 📊 Interactive AI dashboard mockup
- 🏢 Infinite scrolling "Trusted By" logo marquee
- 💬 Testimonial cards with subtle animations
- 🚀 Call-to-Action section
- 🌍 SEO-friendly structure
- ♿ Accessibility-focused design
- ⚡ Optimized for performance

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ss3774-debug/asset-manager.git
```

### Navigate to the project

```bash
cd asset-manager
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
